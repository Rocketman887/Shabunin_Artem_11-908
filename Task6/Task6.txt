 Написать консольное приложение, имитирующее работу веб-приложения (как соц. сеть):
1)В консоли принимаются три вида запросов: /id/100 (любое натуральное число). /feed и /messages. Если запрос неправильный, генерируется файл 404.html (можно открывать файл в веб браузере, джава это умеет достаточно легко).
2) Если запрос правильный, генерируется .html файл, содержащий атрибуты пользователя. В качестве базы данных соответствующий сущностей, можно использовать .txt файлы, можете опционально добавить возможность создания пользователя, новости и сообщений, поставлю баллы.
3) В ответ на /feed генерируется html-страница, хранящая список публичных постов всех пользователей. Данные берутся из файла posts.txt и users.txt. В каждом посте отображается минимум: название, описание, автор, время публикации, текст. Можете добавить атрибуты в сущность поста опционально.
4) Для запроса /messages генерируется html страница, хранящая список всех сообщений пользователей друг другу. Данные берутся из файла messages.txt и users.txt
5) В ответ на каждый запрос гарантированно генерируется отдельный html файл, имя для него можно выбрать произвольным образом
